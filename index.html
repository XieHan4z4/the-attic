The Experience

FIG. 01 // CORE CONCEPT

"Memory is a dusty attic where clarity must be earned through stillness."

The Attic is an immersive sound odyssey that physicalizes the fragmented nature of human recollection. In this near-pitch-black void, participants must navigate through auditory proximity. Adult restlessness is manifested as chaotic static noise, which only subsides when the participant chooses to stand still—allowing the delicate echoes of childhood memories to emerge from the silence.

FIG. 02 // RESEARCH & CONTEXT

The Subconscious Repository

Inspired by John Cage and the concept of Sensory Deprivation, our work explores the attic as a quintessentially psychological space. As the highest and most neglected part of a house, it serves as a metaphor for the deep subconscious.

By enforcing extreme visual restraint, we shift the hierarchy of senses. We force the participant to abandon their ocular-centric habits and rely entirely on an "auditory radar." This sensory shift amplifies the emotional impact of finally "finding" a sound in the void.

FIG. 05 // SYSTEM BLUEPRINTS

Technical Implementation

The core technical challenge of this project was translating the player's physical behavior in a 3D space into dynamic audio mixing parameters in real-time. To achieve this, we built a dual-end communication system based on the OSC (Open Sound Control) protocol.

Unreal Engine 5: Spatial Calculation

UE5 LOGIC: VELOCITY TO OSC
NODE: EVENT TICK
  • Data Transmission: Within the Event Tick, we continuously calculate the distance between the Player Character and the 10 memory objects, alongside the player's current velocity (Velocity -> Vector Length XY). These values are sent to Max/MSP via local port 8000.
  • Interactive Lighting: When a player enters an object's Sphere Overlap, a hidden Point Light is triggered. Using a Map Range Clamped node, the distance data is inversely mapped to the light's intensity.
UE5 LOGIC: DISTANCE TO LIGHT
NODE: MAP RANGE CLAMPED

Max/MSP: Algorithmic Mixing

MAX/MSP: ROUTING & SUPPRESSION
NODE: ZL.SORT / PAK
  • Background Suppression Logic: To emphasize the tranquility of "discovering a memory," we used pak and zl.sort nodes to identify the closest object. As the player approaches it, the system raises the object's volume and simultaneously attenuates the global Background Sound.
  • Speed-Driven Noise Modulation: We tied the received player velocity (/speed) directly to the Modulation and Audio Gain parameters of the background sound. High velocity triggers severe algorithmic modulation, generating an uncomfortable, metallic distortion that instinctively forces the player to slow down.

FIG. 06 // POST-PROJECT REFLECTION

Subverting the Visual Hierarchy

The Attic stands as a rebellion against traditional interactive design. By removing the safety of vision, we invited participants to "listen" to their own past. The project confirms that emotional immersion is most powerful when it requires the user's active, patient participation.
